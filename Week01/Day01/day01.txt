자바 
	객체지향 프로그래밍 언어이다
	1990년대 썬마이크로시스템즈 사는 1990년 초 양방향 TV를 
	만드는 제어 박스의 개발을 위한 그린 프로젝트를 시작한다.
	이 프로젝트가 진행되면서 C++언어로 부족한 부분을 
	C++언어를 기반으로 오크라는 언어를 직접 개발하게 된다.
	제임스 고슬링은 이 오크 언어를 발전시켜 
	자바라는 범용적인 프로그래밍 언어를 개발하였다.

	특징 - 단순하다. (문법이 규칙적)
	         시스템에 독립적이다.(운영체제에 독립적이다.)
	         번역언어이다.

	기술 - JavaSE(Standard Edition) : 자바 표준 버전
	         JavaEE(Enterprise Edition) : 자바 기업 버전
	         JavaME(Mobile Edition) : 자바 소형 버전

	용어 - 소스파일 : .java로 된 것(메모장으로 볼 수 있는건 볼 수 있다)
	         			 (메모장으로 볼 수 없는건 볼 수 없다)
	         컴파일러 : 번역기(문법적인 오류를 잡아낸다.)
			.java를 .class로 만든다
	         바이트코드 : 자바언어에서 소스를 컴파일 하면 생성.
			  (바이트코드의 확장자는 .class)
	         *편집할때는 .java로 해야한다.

	처리순서 - .java ---> 컴파일러 ---> .class --> Interpreter ---> 실행 파일

	플랫폼 - 동작하는 곳(하드웨어 - CPU, 소프트웨어 - 운영체제)

	자바플랫폼 - 자바가상기계(JVM과 자바 응용프로그래밍 인터페이스)
			JVM : PU와 같이 실행할 명령어 집합을 갖는 소프트웨어로 제작된 추상의 기계이다.

	자바 API : 자바 프로그램을 실행하기 위한 각종 클래스 라이브러리
	
	플랫폼에 독립적 : 자바로 한번 작성된 프로그램은 어는 플랫폼에서도 실행될 수 있다.

	JRE (Java Runtime Enviroment)
		자바 API와 자바 가상 기계 그리고 자바 프로그램을 실행하기 위한 
		여러 콤포넌트로 구성된 자바실행환경이다

​	JDK (Java Development Kit)
		자바 언어를 이용하여 프로그램을 개발하기 위한 최소한의 환경
	

자바의 구성
	클래스{
	    메소드(입력){
	        기능 구현 하기
	    }

	    메소드 밖에 기능 구현하면 에러 남
	}

		   출력 결과의 자료 타입	이 형태로 입력
	public static 	void 		main(String[] args) {
         접근제한자	          리턴타입		메소드명 (인수, 매개변수 [] = 문자 배열 형태로 입력을 받는다.) 
			main()메소드 수행하고 결과는 void(없다)이다.
			=수행만 하는 것

	main( )는 문자배열형태의 인수로 입력을 받아서 { ... }안에 실행을 하고
	결과는 리턴은 하지 않고(void) 종료한다.

	main( )함수는 특별한 함수 프로젝트의 시작 부분을 표시하는 메소드

클래스 - 자바 프로그램의 단위
	대소문자를 구분하고 public인 경우 반드시 파일이름과 일치해야 한다.
	하나의 파일에는 여러개의 클래스를 만들 수 있지만
	하지만 public으로 된 클래스는 1개만 가능하다.

패키지 - 폴더

자바 클래스의 구조 - 필드와 메소드로 구성된다.

필드 - 소속 변수(명사)

메소드 - 코드를 실행하는 기능의 단위(동사)

함수와 메소드의 차이점
	함수는 독립적으로 존재
	메소드는 Class에 들어있는 함수.
	공통적으로 코드를 실행하는 기능의 단위

키워드 - 문법적으로 의미 있는 단어

식별자 - 프로그래머가 정의하여 사용하는 이름(클래스명 변수명 메소드명 배열명 등)

변수 - 저장공간, 한가지 타입만 저장할 수 있다.

	작성규칙 - 첫글자는 소문자, 다른 단어가 붙을 경우 그 단어의 첫글자는 대문자로 한다

리터럴 - 소스 코드내에 직접 입력된 값
	- 정수리터럴	8진수 - 0을 붙힌다. 
			16진수 - 0x를 붙힌다.
			ex) 	byte byte_value = 9;//리터럴이 9	10진수
				int int_value = 012;//		8진수
				long long_value = 0xadL;//		16진수 
					a=10, b=11, c=12, d=13, e=14, f=15		
					L : long type이라는 뜻

	- 실수리터럴 5E7 = 5 x 10^7
		     0.12E-5 = 0.12 x 10^-5
		     float는 소숫점 6자리까지 정밀하게 표현 7자리를 반올림
		     double은 소숫점 16자리까지는 정밀하게 출력 17자리를 반올림
			뒷부분은 절삭한다.
