기본형과 참조형

 * 기본형(primitive type)
	byte short int long
	float double
	boolean
	char
	void
   1. 첫 글자가 소문자
   2. 산술 연산이 가능하다.
   3. 비객체 타입이라 null로 초기화할 수 없다.
   4. 데이터에 대해서 참조값(주소)를 저장하는게 아니라 실제 값을 Stack에 직접 저장한다.

	* Wrapper Class
	- 기본형을 클래스로 감싼 형태
	  (비객체인 기본형에 null값을 넣으려면 Wrapper Class로 만든다.)
	1. 첫 글자가 대문자
	2. 산술 연산이 불가능하다.
	3. 매개변수로 객체가 요구될 때 사용한다.
	4. 기본형이 값이 아닌 객체로 저장해야 할 때 사용한다.
	5. 객체간 비교가 필요할 때 사용한다.
	= 기본형을 참조형처럼 사용해야 할 때 Wrapper Class를 사용한다.

 * 참조형(reference type)
   1. 기본형을 제외한 나머지를 참조형이라고 한다.
   2. 참조 값(주소)를 가지는 변수
   3. 클래스 이름을 가지고 만들며 변수의 타입이 기본형이 아니면 참조 변수
	(선언한 자료형이 기본형이 아니면 참조형)
   4. 개인이 만든 클래스도 참조형
   ex) 	Date	      d               =	 new Date();
        클래스이름    참조변수(객체)    = 	참조 값(주소)
       
	= 참조 값을 참조 변수(객체)가 가지고 있다.

 * 배열은 기본형으로도 만들 수 있고, 참조형으로도 만들 수 있다.
   ex) int[] arData = new int[3];
