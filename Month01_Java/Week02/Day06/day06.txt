다차원 배열 : 2차원 이상의 배열
	     자바나 C언어 전부 가로줄부터 채운다
	     행부터 채우고 열로 넘어가서 다시 행을 채우는 것

stack 영역 배열 변수안에 2차원배열의 시작주소가 들어있고
heap에 있는 2차원배열 안에 각 행(1차원배열)의 시작주소가 들어있다.

2차원배열 선언 시 값을 직접 넣을 때 행마다 중괄호를 한번 더 사용해서 구분 {{},{}}

다차원배열을 향상된 for문 사용
- 2차원배열
int[][] ar = new int [2][3];
for(int[] a : ar){
   for(int b : a){		처음에 2차원배열[2][3]
	내용		2를 a에 넣음
   }			그럼 a도 1차원배열인 상태이니까 
}			a를 다시 b에 넣어줌

*가장 중요
가변 배열 : 2차원 이상의 배열을 "배열안에 배열"의 형태로 처리
	   2차원 이상의 다차원 배열을 생성할 때 전체 배열 차수 중 
	   마지막 차수의 길이를 지정하지 않고 나중에 각기 다른 길이의 배열을 
	   생성함으로써 고정된 형태가 아닌 유동적인 배열을 형성
	   ex) int[][] score = new int[5][];
	        score[0] = new int[4];
	        score[1] = new int[3];	   각 열의 길이가 다르다.
	        score[2] = new int[2];
	        score[3] = new int[2];
	        score[4] = new int[3];

기본형 배열 : 배열 각 항목에 직접 값을 가지고 있다.
참조형 배열 : 각 항목에 객체의 주소를 가지고 있다.
		ex) String[] str = new String[3];
		     str[0] = "Java";
		     str[1] = "C++";
		     str[2] = "C#";
		str[0]에는 Heap영역에 만들어진 String Java의 주소가 들어있음
		(Java라는 값이 들어있는게 아님)

배열 복사 : 배열은 한 번 생성하면 크기 변경 불가
	   더 많은 저장 공간이 필요하다면 
	   보다 큰 배열을 새로 만들고 이전 배열로부터 항목 값들을 복사

      방법 1. for문 이용
	 2. System.arrayCopy() 이용
		System.arraycopy(src, srcPos, dest, destPos, length);
		src : 원본 배열
		srcPos : 복사 시작 인덱스(여기부터 복사)
		dest : 복사 받을 배열
		destPos : 복사 받을 곳 시작 인덱스(여기서부터 복사 받을꺼)
		length : srcPos부터 얼만큼 복사할지(3을 주면 srcPos부터 3칸 복사)
