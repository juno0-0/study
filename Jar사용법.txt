Jar만드는 방법
- 프로그램이 없어도 cmd에서 코드를 실행할 수 있도록
   Jar라는 압축파일로 만드는 것
   1. 코드짜고 그 프로젝트 우클릭 export
   2. Java - Runnable JAR file
   3. Launch configuration  = 그 프로젝트의 메인파일(Main)
   4. Expotr destination 브라우저눌러서 저장공간 설정하고
      파일 이름부분에 이름 입력하고 저장 누르고 피니쉬
   5. 사용은 cmd가서 jar있는 폴더경로 찾아가서 
       java -jar 파일명.jar

- dir : 디렉토리(폴더) 목록 보기
- cd.. : 체인지 디렉토리, 현재 위치에서 한칸 앞으로
- cd 폴더명 : 해당 폴더로 이동
